# GT Racing Entry Decision Framework

This academic capstone project explores a stage-gate decision framework for evaluating whether an OEM should adapt an existing production vehicle for entry into a GT racing series.

The current repository contains an early prototype of the modeling logic, including initial data preparation, scoring structure, and gate-based evaluation steps. This is not the final version of the project. The framework, assumptions, and scoring logic are still being refined as the capstone progresses.

## Project Status
Current version: early draft / prototype model

## What this version includes
- Initial synthetic/public-style input structure
- Early stage-gate decision logic
- Preliminary scoring and filtering workflow
- Exploratory model setup for comparing candidate platforms

## Planned next steps
- Refine scoring rules and gate definitions
- Improve variable selection and weighting logic
- Validate intermediate outputs and recommendation flow
- Finalize presentation-ready results and documentation
